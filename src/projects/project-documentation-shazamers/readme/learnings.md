# Learnings: Shazamers Prototype



## Description

Add a detailed explanation of this subsection here.



- Correct handling and setup of the Raspberry Pi and an ESP32 microcontroller, including data transmission over a local Wi-Fi network.

- Simple communication between two devices using UDP without much overhead.

- Working with audio processing, audio input, and audio output on microcontrollers and single-board computers.

- Using external APIs in the context of microcontroller-based projects.

- Handling API requests in JavaScript is significantly easier compared to C++.

- APIs are powerful tools for accessing data, but the project also showed that there is a strong dependency on external services.  
  If an API provider shuts down its service, backup solutions are necessary.

- Displaying text on a TFT screen using an ESP32, including limitations when working with special characters such as umlauts.
