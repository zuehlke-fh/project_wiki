# Timeline: [ -till you make it]

## Not really much to say about milestones honestly

Really most of the time spent on this prototype was on the planning, trying to figure out what the simplest, most effective way to build the device would be.

We knew form the getgo that it would be a simple device, that was part of the point for a 2 person team.

But the more we figured out how much we could simplify our idea for the purpose of making a prototype, it became less and less about hitting and milestones and more about just ... building the thing.

The code for the webpage was written in one day and the device built in the lesson after, bugfixing was done finished on the same day. It has **one** moving part, two if you want to count the cardboard trapdoor. There really wasnt a lot that could go wrong.

But it does its job, and for a cheap prototype id say it does it pretty well. 

    - Project Start:       10. Dez 2025
    - Project Deadline:    06. Feb 2026
    - Prototype completed: 21. Jan 2026
