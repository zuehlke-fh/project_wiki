## Basic Topic Description

Provide an overview of the project here. Explain the purpose and goals of the project.



In this project we developed an interactive prototype that works similar to the app Shazam. The goal of the prototype is to recognize the Input song and play it as well as a small screen for the song. The main focus of the project was the use of different api's.



## Idea and Inspiration

Describe the initial idea and the inspiration behind this project. Share any interesting insights or motivations.



The idea for our project was inspired by the popular app Shazam, which can recognize songs by listening to music. We found it interesting to recreate a similar functionality in our own interactive prototype. Especially the technical challenge of working with audio input and external services was a big motivation for us.
