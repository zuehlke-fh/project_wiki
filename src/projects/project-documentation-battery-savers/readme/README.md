# Idea and Inspiration
It has become common knowledge that you should not keep mobile electronic devices plugged
in indefinitely, specifically mobile phones or tablets. Continuous overcharging of the device can
cause damage to the battery and thus reduce its maximum charge capacity over time.

While
some more modern devices may feature a setting which only permits the device to charge
partially (e.g. stops charging at 80%), some older devices may not have this ability.

The goal of this project is to remedy that shortcoming on older devices, by creating a device that
can externally disconnect the power supply from a mobile device after a sufficient amount of
charge has been acquired.
