# Result: [Der Freundeszug]

## Description
Das finale Ergebnis des Projekts ist ein funktionierender, charaktervoller Prototyp, der die Idee einer symbiotischen Beziehung zwischen zwei Robotern auf spielerische Weise erfahrbar macht. Trotz technischer Herausforderungen konnte das Konzept erfolgreich umgesetzt und durch den kreativen Einsatz verschiedener Materialien und Methoden realisiert werden.

## Content
Der entstandene „Freundeszug“ zeigt zwei miteinander interagierende Einheiten: den Pusher, der aktiv Bewegung steuert, und die Blume, die auf ihr Umfeld reagiert. Gemeinsam bilden sie ein System, das Abhängigkeit, Kommunikation und gegenseitige Unterstützung verkörpert. Durch Lichtsensorik, akustisches Feedback und ein gemeinsames Bewegungssystem entsteht eine einfache, aber symbolisch starke Darstellung von Symbiose.

Technisch überzeugte das Projekt durch eine stabile Funkverbindung zwischen den ESP32‑Boards sowie eine funktionsfähige Sensorsteuerung. Das Gehäuse vereint pragmatische Konstruktion mit kreativem Charakter. :)

Insgesamt bildete das Ergebnis nicht nur einen funktionierenden Roboter‑Prototypen, sondern auch den Abschluss eines Lernprozesses, in dem technische Probleme in kreative Lösungen verwandelt wurden – ein greifbares Beispiel für Teamarbeit, Experimentierfreude und iterative Gestaltung.
