## Basic Topic Description
Provide an overview of the project here. Explain the purpose and goals of the project.

Die Anforderung an unser Projekt war eine Symbiose zu erschaffen.
Unser Ziel:
Zwei seperate "Autos" zu bauen die nicht direkt miteinander kommunizieren, aber auf die Bewegungen des anderen reagieren.

## Idea and Inspiration
Tom & Jerry soll ein Diorama der bekannten gleichnamigen Fernsehsendung sein. Die Maus provoziert die Katze, dann jagt die Katze nach der Maus. Die Maus entkommt aber immer. Die Symbiose entsteht durch die verwendung von Distanzsensoren. Basierend auf deren Werte bewegen sich die Autos in eine Richtung.
