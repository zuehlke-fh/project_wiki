# Result

The final result of *Delamains Dilemma* is a fully functional prototype of a remote-controlled car that intentionally resists user control under certain conditions.

The vehicle successfully combines technical functionality with conceptual intent. Its rebellious behavior is not a malfunction but a core design element.

The prototype represents a symbiotic relationship between user and machine. Instead of enforcing dominance or submission, the system encourages responsible use by making its limitations tangible.

By embedding these constraints into the interaction itself, *Delamains Dilemma* demonstrates how physical interfaces can guide user behavior without relying on explicit instruction or enforcement.
