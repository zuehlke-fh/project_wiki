## Basic Topic Description
Basic Topic Description
Das Projekt untersucht das Thema Symbiose anhand einer Beziehung zwischen zwei Robotern, die aufeinander angewiesen sind, um zu „existieren“. Dabei wurde eine freundschaftliche Form des Zusammenlebens interpretiert, in der technologische Wesen mit unterschiedlichen Bedürfnissen lernen, sich gegenseitig zu unterstützen. Das Zusammenspiel von Nähe, Abhängigkeit und gegenseitiger Fürsorge steht dabei im Mittelpunkt.
## Idea and Inspiration
Idea + Inspiration
Die Grundidee war, eine emotionale Verbindung zwischen zwei autonomen Robotern zu schaffen – dem „Pusher“, der nach Gesellschaft sucht, und der „Blume“, die das Sonnenlicht zum „Überleben“ braucht. Ursprünglich sollte der Pusher seine Freundin mithilfe von Sensorik aktiv finden und ihr helfen, doch technische Limitierungen führten zu einem konzeptionellen Wandel: Stattdessen entstand ein „Freundeszug“, bei dem die Roboter im wörtlichen und übertragenen Sinn gemeinsam unterwegs sind.
Visuell und erzählerisch inspiriert wurde das Projekt von Wall·E, der die Menschlichkeit in Maschinen zeigt, und von Ziemlich beste Freunde, wo gegenseitige Abhängigkeit zu echter Freundschaft führt.
